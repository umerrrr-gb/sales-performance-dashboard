# Sales Performance Dashboard

## Project Overview
This project analyzes sales data to understand business performance using SQL Server and Power BI.

## Tools Used
- SQL Server
- Power BI

## Key Features
- Revenue analysis
- Customer segmentation
- Top products identification
- Regional performance tracking

## Key Insights
- Technology category contributes the highest revenue
- Sales peak during November and December
- East region performs the best among all regions
